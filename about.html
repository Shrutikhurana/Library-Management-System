LIBRARY MANAGEMENT SYSTEM

Library management system deals with all the aspects relating to the efficient working of the library. This software can be used to access all the information relating to the books, authors, publishers etc. A member's personal information can only be accessed by that person only.Records of books borrowed and returned are maintained. This software is dynamic enough to add records of new members and modify the existing members as per the requirements. This can also be used to control the working of staff in the library. This system is compact and efficient in its working and can be used to manage libraries efficiently and effectively.
